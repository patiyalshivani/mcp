# Tool Usage

Use this example once the technical SEO audit MCP server is connected.

## technical_seo_audit

Input:

```json
{
  "url": "https://example.com"
}
```

Prompt:

```text
Run a technical SEO audit for https://example.com and summarize the most important issues.
```

Returns:

- HTTP status
- title and meta description
- headings
- structured data
- internal and external link counts
- DataForSEO technical checks
- page size and response metrics
